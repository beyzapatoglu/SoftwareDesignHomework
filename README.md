# Software Design Homework

This project demonstrates the use of
Factory Pattern and Singleton Pattern
in a campus food ordering system.

## Design Patterns Used

- Factory Pattern
- Singleton Pattern

## Project Structure

- Notification.java
- EmailNotification.java
- SmsNotification.java
- PushNotification.java
- NotificationFactory.java
- AppConfig.java
- CampusFoodOrderService.java

## How to Run

1. Open the project in IntelliJ IDEA
2. Run CampusFoodOrderService.java
3. The output will appear in the console

## Expected Output

The system creates food orders and sends
different notification types such as:
- EMAIL
- SMS
- PUSH notifications
